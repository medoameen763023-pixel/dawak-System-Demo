## 🌟 Key Features

- **Inventory Tracking:** Manage and track stock levels in real-time.
- **User Portals:** Dedicated interfaces for different user roles within the system.
- **Detailed Reporting:** Generate comprehensive reports for operations and sales.
- **User-Friendly Interface:** Intuitive and clean design ensuring smooth user experience.

- **Frontend:** HTML5, CSS3, JS, Bootstrap
- **Database:** MySQL
